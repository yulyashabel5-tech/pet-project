# Смена аватарки в Telegram на Android

## Способ 1. Через **Настройки**

1. Откройте Telegram.

2. Нажмите **Настройки** (⚙️).

3. Нажмите на текущую аватарку (или на пустое место для аватарки) вверху экрана.

4. Выберите источник аватарки: 

    * Галерея — готовое фото.

    * Камера — новое фото.

    * Выбрать эмодзи — анимированная аватарка.

5. Обрежьте аватарку с помощью круглого шаблона. Аватарку можно масштабировать и перемещать.

6. Нажмите **Готово**. Аватарка обновится.

## Способ 2. Через **Профиль**

1. Откройте Telegram.

2. Нажмите **Профиль**.

3. Нажмите **Выбрать фото**.

4. Выполните шаги 4–6 из способа 1.
